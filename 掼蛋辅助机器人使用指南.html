安装指南

系统要求

  • 操作系统：Windows 10/11, macOS 10.15+, Linux
  • Python 3.7 或更高版本
  • 至少 4GB 内存
  • 500MB 可用磁盘空间

安装步骤

1

安装Python

如果您的系统没有安装Python，请从Python官网下载并安装最新版本。

安装时请勾选 "Add Python to PATH" 选项。

2

下载程序文件

下载掼蛋辅助机器人的完整代码文件：

下载完整代码 (ZIP)

解压下载的ZIP文件到您选择的目录。

3

安装依赖库

打开命令提示符(Windows)或终端(macOS/Linux)，导航到程序目录：

cd path/to/guandan-assistant

安装所需的Python库：

pip install -r requirements.txt

主要依赖库：PyQt5, numpy, opencv-python, torch（可选）

4

(可选) 打包为可执行文件

如果您想创建独立的可执行文件：

pip install pyinstaller pyinstaller --onefile --windowed --name GuandanAssistant guandan_assistant.py

打包后的程序位于 dist 目录中

运行程序

启动程序

1

通过Python运行

在程序目录中打开终端/命令提示符，执行：

python guandan_assistant.py
2

运行可执行文件

如果您已打包程序，双击 GuandanAssistant.exe (Windows) 或相应可执行文件。

3

初始设置

首次运行时，程序会自动创建配置文件。您可能需要授予摄像头访问权限（如果使用真实图像识别）。

使用说明

游戏流程

1

开始新游戏

点击 "🆕 新游戏" 按钮初始化游戏

2

扫描手牌

点击 "📷 扫描手牌" 按钮，选择手牌照片或使用摄像头拍摄

💡 提示：将手牌平铺在深色背景上，确保光线充足

3

记录对手出牌

在对手出牌后，在输入框中输入对手出牌（空格分隔）

示例：红桃5 方块5 黑桃5

点击 "📝 记录对手出牌" 按钮

4

获取并选择出牌建议

系统会自动显示多个出牌策略

点击策略按钮查看具体牌

点击 "✅ 采用建议" 自动出牌

或手动选择牌后点击 "🃏 出牌"

5

跳过回合

无法出牌时，点击 "⏭️ 跳过" 按钮

核心功能操作

🔄 更新建议

重新计算出牌策略，提供不同牌型选择（如单张、对子、顺子等）

🔍 识别牌型

选择手牌后点击此按钮，分析所选牌的牌型

📊 游戏状态

实时显示当前轮次、出牌方和剩余牌数

📜 出牌历史

完整记录每轮双方出牌情况

功能特点

核心功能

🖼️ 手牌识别

通过拍照识别手牌，支持真实图像或模拟识别

🧠 智能策略

AI引擎提供多种出牌策略：单张、对子、顺子、炸弹等

🔄 策略轮换

"更新建议"按钮提供不同出牌思路，避免模式化

📈 游戏状态跟踪

实时跟踪轮次、出牌方、剩余手牌等状态

📚 历史记录

完整记录每轮出牌情况，便于复盘分析

🔍 牌型识别

分析所选牌的牌型（单张、对子、顺子、炸弹等）

掼蛋规则支持

程序支持标准掼蛋规则：

  • 牌型识别：单张、对子、顺子（5张以上）、炸弹（4张以上）
  • 牌值排序：2 > A > K > Q > J > 10 > ... > 3
  • 压制规则：相同牌型才能压制，顺子可被更长顺子压制
  • 炸弹可压制任何牌型
  • 先手出牌策略多样化

常见问题

使用问题

❓ 图像识别不准确怎么办？

1. 确保拍照时光线充足，牌面清晰
2. 使用深色背景
3. 尝试手动调整识别结果（后续版本支持）
4. 使用文本输入模式

❓ 为什么建议不出牌？

1. 当前是对手回合
2. 没有能压制对手的牌
3. 策略建议保留大牌
4. 手牌不符合出牌规则

❓ 如何切换不同策略？

1. 点击"更新建议"按钮生成新策略
2. 在策略按钮区域选择不同方案
3. 先手时系统会提供3-5种策略选择

技术问题

⚠️ 安装依赖时出错

尝试以下解决方案：
1. 更新pip: pip install --upgrade pip
2. 使用Python虚拟环境
3. 安装Microsoft Visual C++ Build Tools (Windows)

⚠️ 程序启动失败

常见原因：
1. 缺少依赖库 - 重新运行pip install -r requirements.txt
2. 文件路径包含中文或特殊字符
3. 防病毒软件阻止 - 添加例外

⚠️ 摄像头无法使用

1. 检查系统摄像头权限设置
2. 确保没有其他程序占用摄像头
3. 使用文件选择模式替代实时拍摄

💡 使用提示

• 先手时尝试不同策略按钮，找到最适合当前手牌的出法
• 使用"识别牌型"功能分析复杂牌型
• 大牌和炸弹尽量保留到关键回合使用
• 注意观察历史记录中的对手出牌模式
